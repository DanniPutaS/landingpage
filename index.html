Apa Itu HTML?

HTML (HyperText Markup Language) adalah bahasa pemrograman dasar yang digunakan untuk membangun struktur dan konten halaman web. Bayangkan HTML sebagai kerangka rumah; ia menentukan di mana letak pintu, jendela, dan ruangan-ruangan.

Mengapa HTML Penting?

  • Fondasi Website: Semua halaman web, dari yang paling sederhana hingga yang paling kompleks, dibangun di atas HTML.
  • Mudah Dipelajari: Sintaks HTML relatif mudah dipahami, bahkan bagi pemula.
  • Standar Industri: HTML adalah standar yang diakui secara global, sehingga hampir semua browser web dapat menginterpretasikan kode HTML.

Struktur Dasar HTML

  • !DOCTYPE html : Deklarasi tipe dokumen HTML.
  • html /html : Elemen akar dari dokumen HTML.
  • head /head : Bagian kepala dokumen, berisi informasi meta seperti judul halaman.
  • body /body : Bagian badan dokumen, berisi konten yang akan ditampilkan di halaman.

Element-elment Dasar HTML

  • h1 sampai h6 :Untuk judul dengan berbagai tingkat kepentingan.
  • p :Untuk paragraf.
  • a :Untuk membuat link.
  • img :Untuk menyisipkan gambar.
  • div :ntuk membagi konten menjadi bagian-bagian.
  • span :Untuk menandai bagian teks tertentu.
